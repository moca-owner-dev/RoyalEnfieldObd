# RoyalEnfieldObd

Estructura principal del repositorio:

- `backend/`: API y logica de diagnostico (FastAPI).
- `frontend/`: interfaz web (Vite).
- `scripts/`: scripts auxiliares, pruebas, datos y utilidades.

## Inicio rapido

### Backend

```bash
cd backend
pip install -r requirements.txt
python main.py
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
